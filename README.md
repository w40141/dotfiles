### 2. Homebrew をインストールする.  

1. 以下のコマンドをコピペする.  

		$ ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)"

	>[homebrew](http://brew.sh/index_ja.html)

2. 以下のコマンドでエラーを確認する.  

		$ brew doctor
